Blueprints

Providing a framework to define the API lifecycle in a modular and flexible enough way that anyone can use to define their own version of the API lifecycle using a common set of elements and actions being applied across operations.

About this API Lifecycle Project

This project runs on Github, and the lifecycle blueprints, elements, and actions are defined as YAML, and the website is powered using Jekyll, allowing everything to be updated in a collaborative way using Git.
